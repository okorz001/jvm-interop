# interop-java-lib

A sample Java library demonstrating API-visible features.

## Features

_Italics_ is not implemented yet.

Directly Callable API:

* Static fields
* Static methods
* Constructors
* Instance fields
* Instance methods
* Varargs
* _Generics_

Extensible API:

* Interfaces
* Interfaces with methods
* Base classes
* Annotations
  * Types
  * Constructors
  * Methods
  * Fields
  * Parameters
