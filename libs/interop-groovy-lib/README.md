# interop-groovy-lib

A sample Groovy library demonstrating API-visible features.

## Features

_Italics_ is not implemented yet.

Directly Callable API:

* Static properties
* Static fields
* Static methods
* Static extension methods
* Constructors
* Instance properties
* Instance fields
* Instance methods
* Instance extension methods

Extensible API:

* Closures
* Closures with delegates
* Interfaces
* Traits
* Base classes
* Annotations
  * Types
  * Constructors
  * Methods
  * Fields
  * Parameters
