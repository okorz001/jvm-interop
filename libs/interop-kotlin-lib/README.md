# interop-kotlin-lib

A sample Kotlin library demonstrating API-visible features.

## Features

_Italics_ is not implemented yet.

Directly Callable API:

* Package properties
* Package functions
* Companion properties
* `@JvmStatic` companion properties
* Companion fields
* Companion methods
* `@JvmStatic` companion methods
* Constructors
* Instance properties
* Instance fields
* Instance methods
* Instance extension methods (package functions with receiver)
* Object properties
* `@JvmStatic` object properties
* Object fields
* Object methods
* `@JvmStatic` object methods
* _Data classes_
* _Sealed classes_

Extensible API:

* Functions
* Functions with receivers
* Interfaces
* Base classes
* _Annotations_
