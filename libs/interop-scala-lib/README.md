# interop-scala-lib

A sample Scala library demonstrating API-visible features.

## Features

_Italics_ is not implemented yet.

Directly Callable API:

* Package properties
* Package methods
* Companion properties
* Companion methods
* Constructors
* Instance properties
* Instance methods
* Object properties
* Object methods
* _Case classes_
* Methods with multiple parameters lists

Extensible API:

* Functions
* Curried functions
* Traits
* Traits with properties
* Base classes
